01 / THE HEADLINE

The fast lane is not the hard part.

Most of the set has a documented developer entry point. The operational bottleneck is not finding an API—it’s resolving tenant approval, commercial access, and “public docs versus usable production access.”

Start with the self-serve cohort. Run a short validation queue for admin-bound tools. Treat the gated group as a partner pipeline, not an engineering backlog.

04 / TRUST, NOT THEATER

Verification is a separate product loop.

Every row exposes a first-party starting point. The supplied set is transcribed exactly; triage remains conservative until a browser/MCP worker captures a dated excerpt.

Audit protocol · required before ship

1. Random + stratified 12-app sample

2. Compare four claims against first-party docs

3. Correct row, attach excerpt, rerun metrics

4. Publish accuracy and misses—not just wins

Human-required: tenant administrator consent, paid-plan/partner gates, and ambiguous “API access” wording.
